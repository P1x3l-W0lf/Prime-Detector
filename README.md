# Prime-Detector
A simple site to detect whether or not a number is prime. Live [here](https://P1x3l-W0lf.github.io/Prime-Detector/).

## About This Project
This is a simple script to detect whether or not a number (or list of them) is prime, given a UI.

You can add as many numbers (less than 9,000,000,000,000,000) as you'd like to the list, separated by spaces or comas, and you will be told which of them if any are prime numbers.

The max is 9,000,000,000,000,000 because that is the tidiest number before JavaScript goes into BigInt territory. The cutoff is technically closer to 9,007,197,427,937,536 but 9 and a bunch of zeros is more visually pleasing.
